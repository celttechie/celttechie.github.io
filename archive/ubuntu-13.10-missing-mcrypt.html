PHP missing mcrypt.ini file in Ubuntu 13.10

Date Tue 08 April 2014 Tags Ubuntu / 13.10 / mcrypt / PHP

So doing a dist-upgrade to Ubuntu 13.10 resulted in mcrypt no longer being loaded in PHP. The fix?

ln -s /etc/php5/conf.d/mcrypt.ini /etc/php5/apache2/conf.d/20-mcrypt.ini
service apache2 restart
